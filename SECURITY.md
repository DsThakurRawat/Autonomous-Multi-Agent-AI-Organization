# Security Policy

## Supported Versions

The following versions of the Autonomous Multi-Agent AI Organization are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of our project very seriously. If you discover a vulnerability, please report it privately:

1. Navigate to the **Security** tab of the GitHub repository.
2. Click on **Report a vulnerability**.
3. Provide a clear description of the issue, steps to reproduce it, and the potential impact.

Alternatively, you can contact the repository owners directly. Please **do not** report security vulnerabilities via public GitHub issues.

## Scope
This policy applies to all core components of the Autonomous Multi-Agent AI Organization, including but not limited to:
- The core routing and orchestration logic.
- Backend services and components.
- Integrated multi-agent logic and protocols.
